This theme is meant specifically for College of Charleston 

== Installation ==

1. Upload the theme folder to the `/wp-content/themes/` directory or install from the WordPress dashboard.
2. Activate the theme through the ‘Appearance > Themes’ menu in WordPress.
3. Customize the theme via the WordPress Customizer (Appearance > Customize).
